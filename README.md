# Flight Management Frontend v3
Test commit from new setup

React + Material-UI frontend for Flight Booking.

## Features in v3
- Search flights by number or destination (sample data included).
- Click a flight row to select it for booking.
- Passenger details form with DOB calendar, gender radio, seat-class radio,
  number of passengers with +/- controls, and live fare calculation per class.
- Class pricing multipliers:
  - First Class: +70%
  - Business: +50%
  - Premium Economy: +30%
  - Economy: base price

## Quick start

1. Extract the zip.
2. `cd flight-frontend-v3`
3. `npm install`
4. `npm start`
